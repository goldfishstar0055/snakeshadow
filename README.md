# ヘビ日陰マップ 🗺️

「ヘビの出ない日陰で過ごしたい」を叶える、ArcGIS 3Dマップのブラウザアプリです。

**デモ**: https://goldfishstar0055.github.io/snakeshadow/

## 機能

- **3D日陰表示**: 現在の太陽位置に基づく建物・地形の影をリアルタイム表示
- **天気連動**: Open-Meteo APIで現在の天気・雲量を取得し、曇り/雨の場合は全面日陰として表示
- **ヘビ出現ピン**: Googleフォーム回答シートからデータを取得し、赤ピンで表示（クリックで詳細）
- **目撃情報の報告**: 地図上でタップした地点の座標をGoogleフォームに自動入力して報告
- **プレビューモード**: 日付・時刻・天気を自由に変えて影のシミュレーション
- **PWA対応**: ホーム画面に追加してアプリのように全画面で使用可能

## ヘビ目撃を報告するには

1. 画面の「🐍 目撃を報告」ボタンをタップ
2. 地図上で目撃した場所をタップ → 青いピンが立つ
3. 「この地点で報告」をタップ → Googleフォームが開く（緯度経度は自動入力済み）
4. フォームで日付・場所・状況などを入力して送信
5. 次回のデータ読み込み時にピンとして反映されます

## ホーム画面に追加（PWA）

### iPhone (Safari)
1. SafariでデモURLを開く
2. 共有ボタン（□↑）→「ホーム画面に追加」
3. アプリのように全画面で起動します

### Android (Chrome)
1. Chromeでデモを開く
2. メニュー（⋮）→「ホーム画面に追加」または「アプリをインストール」
3. ホーム画面から起動できます

## ローカル開発

```bash
npm install
cp .env.example .env
# .env に ArcGIS API キーを設定
npm run dev
```

## 技術スタック

- Vite + Vanilla JavaScript + vite-plugin-pwa
- ArcGIS Maps SDK for JavaScript（3D SceneView）
- Open-Meteo API（天気・雲量取得、APIキー不要）
- Googleスプレッドシート（フォーム回答シート）からCSV取得

## データソース

ヘビ出現データはGoogleフォームの回答シートから取得しています。
フォーム送信後、アプリの「データ再読み込み」ボタンで最新データを反映できます。
